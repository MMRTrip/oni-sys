# Oni-Sys: Демон минимализма

> **Oni-Sys** — это демон-оператор, призванный служить своему хозяину. Он живёт внутри Arch-based системы, следит за порядком оперативной памяти, отображает состояние железа в виде RPG-панелей и меняет свою внешность в зависимости от задачи.
>
> Его команды — не сухие утилиты, а древние ритуалы: `cast` вместо установки пакетов, `levelup` вместо обновления, `hunter` для охоты на прожорливые процессы.

---

## Философия проекта

Oni-Sys создан как оптимизированная оболочка для слабого железа (*4 ГБ RAM*). Мы отказались от лишних украшательств в пользу производительности и единого визуального языка.

Ключевые принципы:
- Минимализм без компромиссов.
- Единая цветовая палитра (#ff2a5f) во всех слоях интерфейса.
- Система бэкапов со списком установленных пакетов для полного восстановления.
- Концепция «ритуалов» вместо команд.

---

## Структура репо

| Папка | Описание |
|------|----------|
| `/autostart/` | Системные алерты и автозапуск демонов мониторинга ресурсов. |
| `/fish/functions/` | Кастомные Fish-функции *(RPG-команды)* и скрипты-бэкапов (`oni-*`). |
| `/kde/` | Полная конфигурация KDE Plasma: панели, горячие клавиши, динамические обои с погодой. |
| `/starship/ & /konsole/` | Темы терминала Starship и Yakuaке с единой цветовой ДНК. |
| `/packages/` | Паспорт твоей системы: списки всех установленных пакетов (*pacman/AUR/Flatpak*) и оверрайды Flatpak. |
| `/sizer/*` | Критические системные файлы: тема SDDM, GRUB, настройки ядра (*zram*, *sysctl*), разметка рабочего стола. |
| `/system/` | Конфигурация загрузчика GRUB и менеджера дисплея SDDM. |
| `/yakuake/` | Настройки эмулятора терминала Yakuake: полупрозрачное окно всегда открыто поверх всех приложений. |
| `/PKGBUILD/` | PKGBUILD-файл для AUR. Позволяет установить всю систему одним пакетом. |

## Скриншоты и примеры

*(Здесь ты можешь добавить свои скриншоты рабочего стола, терминала и панелей мониторинга).* 

Например:
```
$ oni-fetch # Наш вылизанный фетч с HP оперативки
$ oni-status # Панель состояния системы в стиле RPG
$ vbi-burst # Запустить музыку из плеера
$ oni-hunter # Убить самый прожорливый процесс
```

## Установка

Для полной интеграции рекомендуется использовать пакет из AUR:
```bash
yay -S oni-system-config
```

Или можно вручную развернуть конфиги по директориям:
```bash
git clone https://github.com/<your_username>/oni-sys.git ~/.config/oni-sys
./oni-sys/fish/functions/oni-backup.fish --restore
```
*(Не забывай про права sudo при восстановлении системных файлов!).*
